## 📖 À propos du projet

Bienvenue sur le dépôt du projet de site de vente de fleurs. Ce projet a pour but de proposer une vitrine élégante pour des plantes ornementales et des compositions florales, en mettant en avant le savoir-faire **"Made in DRC"**.

## ✨ Fonctionnalités actuelles

| Fonctionnalité | Description | Statut |
| :--- | :--- | :---: |
| 🌸 **Catalogue de produits** | Présentation des fleurs (Rose Rouge, Tulipe bleue) avec prix | ✅ |
| 🔍 **Barre de recherche** | Pour trouver facilement une plante ou une composition | ✅ |
| 🎨 **Interface UI** | Structure en HTML/CSS prête à être dynamisée | 🚧 |

## 🛠 Technologies utilisées

*   **HTML5** : Structure sémantique du site
*   **CSS3** : Mise en forme et design

## 🚀 Comment lancer le projet

Il s'agit d'un site web frontend statique. Pour le visualiser localement :

1. Clonez ce dépôt sur votre machine :
   ```bash
   git clone <votre-lien-github-ici>
   ```
2. Ouvrez le fichier `index.html` dans votre navigateur web préféré.
